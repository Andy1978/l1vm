FANN neural network tests, XOR data and training set.
=====================================================

To compile FANN libary XOR demo:
$ comp/l1com lib/fann-lib
$ assemb/l1asm lib/fann-lib

To run the XOR demo:
$ vm/l1vm lib/fann-lib

The data and neural network is in fann/ directory.
The training source code can be compiled by running:
$ ./make.sh
